Data Access & Integration

Your mission data, simplified.

The CASSI Integrated User Interface (IUI) simplifies configuration and interaction with the module, providing a unified environment for system control and data analysis. This UI makes it easier to visualize information, monitor the camera feed, and test specific features such as force calibration and mode switching.

Six Dedicated Panels

  • 3D Visualization of quaternion data (powered by Three.js)
  • Graph display of quaternion data (using Chart.js)
  • Terminal output showing received data
  • Live camera view
  • Quaternion and system status overview
  • Control panel with interactive buttons

Why it Matters

Whether you’re operating a single CubeSat or an entire constellation, the IUI offers a unified dashboard for both real-time and historical data, customizable views for mission-specific needs, and seamless integration with your ADCS software. Secure cloud access, offline capabilities, and built-in visualization tools ensure your team can make informed decisions without technical bottlenecks.
